# pruebaagustin

Landing page para infoproductores, construida en Astro, inspirada en la estructura de:

- https://www.psicoemprendedores.org/webinar2025-fm
- https://tulegadoderiqueza.com/conoce-a-ramiro/

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

## Estructura principal

- `src/pages/index.astro`: landing principal
- `home.html`: versión HTML original usada como base
